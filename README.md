# iPhone Classic Web App для Google Apps Script

Это веб-приложение разворачивается в Google Apps Script, привязанном к Google Sheets. Оно имитирует внешний вид iPhone первого поколения и добавляет простые интерактивные экраны: телефон, SMS, Safari, заметки, iPod и настройки.

## Что нужно подготовить

1. Создайте Google Sheet и добавьте лист `Notes`.
2. В первой строке листа `Notes` добавьте заголовки: `Title` и `Body`.
3. Со второй строки можно добавлять заметки.

## Как развернуть

1. Откройте Google Sheets → `Расширения` → `Apps Script`.
2. Замените содержимое файла `Code.gs` и добавьте файл `Index.html` (название должно совпадать).
3. Добавьте файл `appsscript.json` (в редакторе включите отображение `appsscript.json`).
4. Нажмите `Развернуть` → `Новое развертывание` → `Веб-приложение`.
5. Выберите доступ: `Любой`.
6. Сохраните и откройте выданную ссылку.

## Как пользоваться

- Нажимайте на иконки приложений, чтобы открыть экран.
- В приложении `Заметки` нажмите `Обновить` — данные подтянутся из Google Sheets.

## Кастомизация

- Обои и цвета регулируются в `Index.html` (CSS в начале файла).
- Можно добавить больше экранов, повторяя структуру `.app-screen`.
