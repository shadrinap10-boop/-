Проект — веб‑приложение для частного центра «Я САМ», где родители видят услуги, цены и расписание и могут забронировать конкретный слот занятия через понятную форму.​ Система включает публичный интерфейс в фирменной детской стилистике и админ‑часть для управления контентом и заявками.
​Основная идея:
Сделать единое “окно” для выбора занятий: от просмотра программ и цен до выбора времени и отправки заявки на конкретное занятие.
Акцент — на простом пути записи (минимум шагов) и аккуратном фирменном UI, соответствующем бренду «Я САМ».
Цели проекта:
Упростить родителям поиск подходящего курса и времени занятия через структурированный каталог и расписание.
Снизить нагрузку на администратора за счёт автоматизации заявок и уведомлений.
Обеспечить управляемость: администратор меняет услуги/слоты/статусы заявок и выгружает данные для отчётности.
Функционал:
Публичная часть:
Каталог услуг: карточки, детальная страница услуги (описание, цены, возраст/уровень, ближайшие слоты).
Расписание: просмотр доступных слотов по возрастным группам/классам.
Бронирование слота: выбор конкретного времени + форма (ФИО родителя, ФИО ребёнка, возраст, телефон, комментарий) + подтверждение заявки.
Админ‑часть:
Авторизация админа и панель управления.
Управление услугами и расписанием, обработка заявок (статусы, заметки), возможность добавления услуг, управление возратсными группами, ценами, расписанием.
Экспорт заявок в Excel для отчётности/учёта.
Интеграции и данные:
Уведомление администратору в Telegram при создании заявки.
Импорт услуг/цен и расписания из Excel
